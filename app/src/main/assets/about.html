Camera2Test

This application analyzes the Camera2 API support of your device.
It retrieves tha available Camera2 API keys and shows the values according to the analyzed camera.
It is useful mostly for Android developers and testers to get known the available features on their devices.
You can share your Camera2 API result with your team members.
